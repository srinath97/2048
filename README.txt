PROBLEM: 2048 is a game consisting of 16 tiles with 4 rows and 4 columns. When 2 number touch each other then they merge. You win when you get 2048. 
You can move a tile by using arrow keys. You lose when you cannot move any tail. 

This program works in DEV-C++ version 5.8.3.
